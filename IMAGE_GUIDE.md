## üîÑ HOW TO REPLACE IMAGES

### **Method 1: Direct File Replacement**
1. Save your photo with a descriptive name (e.g., `hero-team-cleaning.jpg`)
2. Place in `public/images/` folder
3. Update the `src` attribute in the code:
   ```jsx
   // Change from:
   src="https://images.unsplash.com/photo-..."
   
   // To:
   src="/images/hero-team-cleaning.jpg"
   ```

### **Method 2: Using Image Hosting (Recommended)**
1. Upload to **Cloudinary**, **Imgix**, or **AWS S3**
2. Optimize images for web (compress, resize)
3. Get CDN URL
4. Replace `src` URLs in code

### **File Locations to Update:**
- `src/pages/HomePage.tsx` - Lines 13, 111, 137, 163
- `src/pages/ServicesPage.tsx` - Lines 143-145
- `src/pages/AboutPage.tsx` - Lines 60, 75, 91, 107
- `src/pages/ContactPage.tsx` - Line 6

## üìù IMAGE OPTIMIZATION TIPS

**Before uploading:**
1. **Resize** to exact dimensions needed
2. **Compress** using TinyPNG or ImageOptim (60-80% quality)
3. **Rename** descriptively (`team-photo-2024.jpg`)
4. **Add alt text** for accessibility and SEO
5. **Test** on mobile and desktop

**Tools:**
- **Compression:** tinypng.com, squoosh.app
- **Resizing:** Photoshop, GIMP, Canva
- **Hosting:** Cloudinary (free tier), Imgix, AWS S3
